# Octopunk Documentation

Welcome to the Octopunk documentation. Octopunk is a modern, keyboard-driven GitHub client built for developers who want a faster, more efficient way to navigate the GitHub platform.

## Getting Started

New to Octopunk? Start here:

- [Getting started](/docs/getting-started) - Download and install Octopunk

## What is OctoPunk?

OctoPunk is a full-fledged GitHub desktop client with AI assistance integrated into the experience.

The project started, as some of the best projects do, fueled by frustration with the day-to-day use of GitHub's web platform.

Over the past few years, there has been a convergence on fast, useful, and most importantly, developer-focused application UX and UI. Fuzzy searching, keyboard shortcuts, theming, and search at your fingertips have all become commonplace in our editors and other modern applications.

Until now, there has been no notable effort to bring these modern workflows to the GitHub platform, let alone in a desktop application.

OctoPunk sets out to remedy this by taking the UX and UI patterns developers love and reimagining them within a GitHub client. Along the way, OctoPunk makes many improvements to using the GitHub platform, too many to list here.

We encourage anyone who is sick of being stuck in the browser, forgets which tab had that one PR they were working on, or simply wants an AI-first GitHub experience to try OctoPunk.
We've built this as developers who work in GitHub every day and know firsthand what a faster, more focused client can do for your workflow.

Octopunk provides a familiar but enhanced issues and pull request UI.

<video src="/screens/octopunk-issues-and-pulls.mp4" controls width="100%" loop muted></video>

An outline is provided on the left drawer of both issues and pull request pages.
This allows you to quickly jump to interesting events within the issue or pull.

Pull requests support the familiar code review flows GitHub has pioneered including
submitting reviews, resolving comment threads, and merging the pull requests.

The diff viewer supports filtering by commit, supporting commit-wise reviews.

The issues pages support sub-issues and issue relationships.
